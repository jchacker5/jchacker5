COMP390 Meteorite Filtering Program

This program allows you to filter a dataset of meteorite landings based on either the mass (in grams) or the year the meteorite fell. Users can specify the range for either mass or year (in YYYY format) to filter the meteorites. The program offers various options to view and save the filtered results.

COMP350 myShell

This project aims to develop a simple UNIX shell (command interpreter) in Java. The shell supports various built-in commands, providing a simple yet effective interface for file and text manipulation.
